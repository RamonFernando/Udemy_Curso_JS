Métodos de Arrays en PHP

Manipulación de elementos

array_push(array, elemento): Agrega uno o más elementos al final del array.

$numeros = [1, 2, 3];
array_push($numeros, 4); // [1, 2, 3, 4]

array_pop(array): Elimina y devuelve el último elemento del array.

$numeros = [1, 2, 3];
$eliminado = array_pop($numeros); // $eliminado: 3, $numeros: [1, 2]

array_unshift(array, elemento): Agrega uno o más elementos al inicio del array.

$numeros = [2, 3];
array_unshift($numeros, 1); // [1, 2, 3]

array_shift(array): Elimina y devuelve el primer elemento del array.

$numeros = [1, 2, 3];
$eliminado = array_shift($numeros); // $eliminado: 1, $numeros: [2, 3]

array_splice(array, posición, cantidad, elementos): Modifica el array eliminando, reemplazando o insertando elementos.

$numeros = [1, 2, 3, 4];
array_splice($numeros, 1, 2, [99]); // [1, 99, 4]

Acceso y búsqueda

array_search(valor, array): Devuelve el índice o clave del primer elemento encontrado; false si no existe.

$numeros = [1, 2, 3];
$indice = array_search(2, $numeros); // 1

in_array(valor, array): Verifica si el array contiene un elemento (devuelve true o false).

$numeros = [1, 2, 3];
$existe = in_array(2, $numeros); // true

Iteración

foreach: Itera sobre cada elemento del array.

$numeros = [1, 2, 3];
foreach ($numeros as $num) {
    echo $num;
}

Filtrado y transformación

array_map(función, array): Crea un nuevo array aplicando una función a cada elemento.

$numeros = [1, 2, 3];
$dobles = array_map(fn($num) => $num * 2, $numeros); // [2, 4, 6]

array_filter(array, función): Crea un nuevo array con los elementos que cumplen una condición.

$numeros = [1, 2, 3, 4];
$pares = array_filter($numeros, fn($num) => $num % 2 === 0); // [2, 4]

array_reduce(array, función, valorInicial): Reduce el array a un único valor acumulado.

$numeros = [1, 2, 3, 4];
$suma = array_reduce($numeros, fn($acumulado, $num) => $acumulado + $num, 0); // 10

Orden y manipulación avanzada

sort(array): Ordena los elementos del array.

$numeros = [3, 1, 4];
sort($numeros); // [1, 3, 4]

array_reverse(array): Invierte el orden de los elementos.

$numeros = [1, 2, 3];
$invertido = array_reverse($numeros); // [3, 2, 1]

array_merge(array1, array2): Combina dos o más arrays.

$numeros1 = [1, 2];
$numeros2 = [3, 4];
$combinado = array_merge($numeros1, $numeros2); // [1, 2, 3, 4]

implode(separador, array): Convierte los elementos del array en una cadena, separados por el valor dado.

$numeros = [1, 2, 3];
$texto = implode('-', $numeros); // "1-2-3"

Clonación y creación

array_slice(array, inicio, longitud): Crea una copia de una porción del array.

$numeros = [1, 2, 3, 4];
$subArray = array_slice($numeros, 1, 2); // [2, 3]

is_array(variable): Verifica si una variable es un array.

$numeros = [1, 2, 3];
$esArray = is_array($numeros); // true
