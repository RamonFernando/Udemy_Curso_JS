Métodos de Arrays en JavaScript

Manipulación de elementos

push(elemento): Agrega uno o más elementos al final del array.

let numeros = [1, 2, 3];
numeros.push(4); // [1, 2, 3, 4]

pop(): Elimina y devuelve el último elemento del array.

let numeros = [1, 2, 3];
let eliminado = numeros.pop(); // eliminado: 3, array: [1, 2]

unshift(elemento): Agrega uno o más elementos al inicio del array.

let numeros = [2, 3];
numeros.unshift(1); // [1, 2, 3]

shift(): Elimina y devuelve el primer elemento del array.

let numeros = [1, 2, 3];
let eliminado = numeros.shift(); // eliminado: 1, array: [2, 3]

splice(posición, cantidad, elementos): Modifica el array eliminando, reemplazando o insertando elementos.

let numeros = [1, 2, 3, 4];
numeros.splice(1, 2, 99); // [1, 99, 4]

Acceso y búsqueda

indexOf(elemento): Devuelve el índice de la primera aparición de un elemento; -1 si no existe.

let numeros = [1, 2, 3];
let indice = numeros.indexOf(2); // 1

lastIndexOf(elemento): Devuelve el índice de la última aparición de un elemento.

let numeros = [1, 2, 3, 2];
let indice = numeros.lastIndexOf(2); // 3

includes(elemento): Verifica si el array contiene un elemento (devuelve true o false).

let numeros = [1, 2, 3];
let existe = numeros.includes(2); // true

Iteración

forEach(función): Ejecuta una función por cada elemento del array.

let numeros = [1, 2, 3];
numeros.forEach(num => console.log(num));

map(función): Crea un nuevo array con el resultado de aplicar una función a cada elemento.

let numeros = [1, 2, 3];
let dobles = numeros.map(num => num * 2); // [2, 4, 6]

Filtrado y transformación

filter(función): Crea un nuevo array con los elementos que cumplen una condición.

let numeros = [1, 2, 3, 4];
let pares = numeros.filter(num => num % 2 === 0); // [2, 4]

reduce(función, valorInicial): Reduce el array a un único valor acumulado.

let numeros = [1, 2, 3, 4];
let suma = numeros.reduce((acumulado, num) => acumulado + num, 0); // 10

find(función): Devuelve el primer elemento que cumple con la condición.

let numeros = [1, 2, 3, 4];
let encontrado = numeros.find(num => num > 2); // 3

findIndex(función): Devuelve el índice del primer elemento que cumple con la condición.

let numeros = [1, 2, 3, 4];
let indice = numeros.findIndex(num => num > 2); // 2

Orden y manipulación avanzada

sort(función): Ordena los elementos del array.

let numeros = [3, 1, 4];
numeros.sort((a, b) => a - b); // [1, 3, 4]

reverse(): Invierte el orden de los elementos.

let numeros = [1, 2, 3];
numeros.reverse(); // [3, 2, 1]

concat(array2): Combina dos o más arrays.

let numeros1 = [1, 2];
let numeros2 = [3, 4];
let combinado = numeros1.concat(numeros2); // [1, 2, 3, 4]

join(separador): Convierte los elementos del array en una cadena, separados por el valor dado.

let numeros = [1, 2, 3];
let texto = numeros.join('-'); // "1-2-3"

Clonación y creación

slice(inicio, fin): Crea una copia de una porción del array.

let numeros = [1, 2, 3, 4];
let subArray = numeros.slice(1, 3); // [2, 3]

Array.from(iterable): Crea un array a partir de un iterable.

let cadena = '123';
let numeros = Array.from(cadena); // ['1', '2', '3']

Array.isArray(objeto): Verifica si una variable es un array.

let numeros = [1, 2, 3];
let esArray = Array.isArray(numeros); // true
