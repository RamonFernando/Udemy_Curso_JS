Métodos de Arrays en Python

Manipulación de elementos

append(elemento): Agrega un elemento al final de la lista.

numeros = [1, 2, 3]
numeros.append(4)  # [1, 2, 3, 4]

pop(): Elimina y devuelve el último elemento de la lista.

numeros = [1, 2, 3]
eliminado = numeros.pop()  # eliminado: 3, lista: [1, 2]

insert(posición, elemento): Inserta un elemento en una posición específica.

numeros = [1, 3]
numeros.insert(1, 2)  # [1, 2, 3]

remove(elemento): Elimina la primera aparición de un elemento.

numeros = [1, 2, 3]
numeros.remove(2)  # [1, 3]

extend(iterable): Extiende la lista añadiendo elementos de otro iterable.

numeros = [1, 2]
numeros.extend([3, 4])  # [1, 2, 3, 4]

Acceso y búsqueda

index(elemento): Devuelve el índice de la primera aparición de un elemento.

numeros = [1, 2, 3]
indice = numeros.index(2)  # 1

in: Verifica si un elemento está en la lista.

numeros = [1, 2, 3]
existe = 2 in numeros  # True

Iteración

for: Itera sobre cada elemento de la lista.

numeros = [1, 2, 3]
for num in numeros:
    print(num)

Filtrado y transformación

map(función, iterable): Crea un nuevo iterable aplicando una función a cada elemento.

numeros = [1, 2, 3]
dobles = list(map(lambda x: x * 2, numeros))  # [2, 4, 6]

filter(función, iterable): Filtra elementos que cumplen una condición.

numeros = [1, 2, 3, 4]
pares = list(filter(lambda x: x % 2 == 0, numeros))  # [2, 4]

reduce(función, iterable): Reduce el iterable a un único valor acumulado.

from functools import reduce

numeros = [1, 2, 3, 4]
suma = reduce(lambda x, y: x + y, numeros)  # 10

Orden y manipulación avanzada

sort(): Ordena los elementos de la lista en su lugar.

numeros = [3, 1, 4]
numeros.sort()  # [1, 3, 4]

sorted(iterable): Devuelve una nueva lista ordenada.

numeros = [3, 1, 4]
ordenados = sorted(numeros)  # [1, 3, 4]

reverse(): Invierte el orden de los elementos.

numeros = [1, 2, 3]
numeros.reverse()  # [3, 2, 1]

+ (concatenación): Combina dos o más listas.

numeros1 = [1, 2]
numeros2 = [3, 4]
combinado = numeros1 + numeros2  # [1, 2, 3, 4]

'separador'.join(lista): Convierte los elementos de una lista en una cadena.

numeros = ['1', '2', '3']
texto = '-'.join(numeros)  # "1-2-3"

Clonación y creación

list(slice): Crea una copia o sublista.

numeros = [1, 2, 3, 4]
sub_lista = numeros[1:3]  # [2, 3]

isinstance(variable, list): Verifica si una variable es una lista.

numeros = [1, 2, 3]
es_lista = isinstance(numeros, list)  # True
