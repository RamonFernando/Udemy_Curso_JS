Métodos de Arrays en Java

Manipulación de elementos

Agregar elementos a una lista: Usando add(elemento) en una lista.

import java.util.ArrayList;
import java.util.List;

List numeros = new ArrayList<>();
numeros.add(1); // [1]
numeros.add(2); // [1, 2]
numeros.add(3); // [1, 2, 3]

Eliminar elementos: Usando remove().

numeros.remove(1); // Elimina el elemento en el índice 1. Lista: [1, 3]

Reemplazar un elemento: Usando set(index, elemento).

numeros.set(1, 99); // [1, 99]

Acceso y búsqueda

Obtener un elemento: Usando get(index).

int numero = numeros.get(0); // 1

Buscar un elemento: Usando indexOf(elemento).

int indice = numeros.indexOf(99); // 1

Comprobar si existe un elemento: Usando contains(elemento).

boolean existe = numeros.contains(99); // true

Iteración

Usando un bucle for:

for (int num : numeros) {
    System.out.println(num);
}

Usando forEach:

numeros.forEach(num -> System.out.println(num));

Filtrado y transformación

Filtrar elementos: Usando streams.

import java.util.stream.Collectors;

List pares = numeros.stream()
                             .filter(num -> num % 2 == 0)
                             .collect(Collectors.toList()); // [2, 4]

Transformar elementos: Usando map.

List dobles = numeros.stream()
                              .map(num -> num * 2)
                              .collect(Collectors.toList()); // [2, 4, 6]

Orden y manipulación avanzada

Ordenar: Usando sort.

import java.util.Collections;

Collections.sort(numeros); // Ordena en orden ascendente.

Invertir el orden: Usando reverse.

Collections.reverse(numeros); // Invierte el orden.

Combinar listas: Usando addAll.

List otrosNumeros = new ArrayList<>();
otrosNumeros.add(4);
numeros.addAll(otrosNumeros); // [1, 2, 3, 4]

Clonación y creación

Copiar un array: Usando Arrays.copyOf.

import java.util.Arrays;

int[] numeros = {1, 2, 3, 4};
int[] copia = Arrays.copyOf(numeros, numeros.length);

Convertir un array a lista: Usando Arrays.asList.

List lista = Arrays.asList(1, 2, 3);

Verificar si una colección está vacía: Usando isEmpty.

boolean estaVacia = numeros.isEmpty(); // false

Obtener tamaño: Usando size.

int tamaño = numeros.size(); // 4
