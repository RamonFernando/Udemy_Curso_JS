Curso de JavaScript

Módulo 1: Fundamentos de JavaScript

Contenido:

  • Introducción a JavaScript y su historia.
  • Configuración del entorno de desarrollo.
  • Variables y constantes: var, let, y const.
  • Tipos de datos primitivos: string, number, boolean, null, undefined.
  • Operadores básicos: aritméticos, lógicos y relacionales.
  • Estructuras de control: condicionales (if, else, switch).
  • Prácticas iniciales para consolidar conceptos básicos.
  • Buenas prácticas para nombrar variables y escribir código legible.

Nivel (1-100): 10

Categoría: Principiante

Descripción: Aprende los conceptos básicos de JavaScript para empezar a programar. Este módulo establece las bases para trabajar con el lenguaje.

Módulo 2: Manipulación de datos

Contenido:

  • Creación y manipulación de arrays: push, pop, shift, unshift.
  • Métodos avanzados de arrays: map, filter, reduce.
  • Creación y manipulación de objetos.
  • Iteración sobre arrays y objetos con bucles como for, for...of, y for...in.
  • Manejo de estructuras anidadas (arrays y objetos dentro de objetos).
  • Transformación de datos en diferentes formatos (JSON).

Nivel (1-100): 30

Categoría: Principiante-Intermedio

Descripción: Aprende a manejar y transformar datos usando estructuras comunes de JavaScript. Este módulo es clave para gestionar información en aplicaciones.

Módulo 3: Programación Orientada a Objetos (POO)

Contenido:

  • Creación de clases y objetos.
  • Herencia y reutilización de código con extends.
  • Encapsulación: modificadores de acceso (private, protected, public).
  • Uso de super para invocar métodos del padre.
  • Composición de objetos y arrays de objetos.
  • Métodos estáticos y su utilidad en proyectos reales.
  • Polimorfismo y sobreescritura de métodos.

Nivel (1-100): 50

Categoría: Intermedio

Descripción: Aprende a estructurar código de forma eficiente y reutilizable con los principios de POO. Este módulo te prepara para proyectos más grandes y organizados.

Módulo 4: Manipulación del DOM

Contenido:

  • Qué es el DOM y cómo interactuar con él.
  • Selección de elementos: getElementById, querySelector, querySelectorAll.
  • Manipulación de contenido HTML y estilos CSS desde JavaScript.
  • Creación y eliminación de elementos: createElement, appendChild, removeChild.
  • Manejo de eventos: addEventListener, eventos de teclado y ratón.
  • Creación de formularios dinámicos y validación de datos.
  • Introducción a la manipulación del modelo de cajas (CSS Box Model) desde JavaScript.

Nivel (1-100): 60

Categoría: Intermedio

Descripción: Aprende a interactuar con el contenido HTML y crear experiencias dinámicas en la web. Este módulo es clave para desarrollar interfaces de usuario.

Módulo 5: JavaScript Moderno (ES6+)

Contenido:

  • Declaraciones let y const.
  • Funciones flecha (arrow functions).
  • Plantillas de cadenas (template literals).
  • Desestructuración de arrays y objetos.
  • Promesas: then, catch, finally.
  • Uso de async y await para manejar asincronía.
  • Operadores spread y rest (...).
  • Introducción a los módulos: import y export.

Nivel (1-100): 70

Categoría: Intermedio-Alto

Descripción: Aprende las características más recientes y poderosas de JavaScript. Este módulo te permite escribir código más limpio y moderno.

Módulo 6: Aplicaciones Dinámicas

Contenido:

  • Almacenamiento de datos en el navegador: localStorage y sessionStorage.
  • Uso de fetch para realizar solicitudes HTTP.
  • Consumo de APIs externas y manejo de datos JSON.
  • Gestor de errores para solicitudes HTTP fallidas.
  • Creación de aplicaciones completas que integren el DOM y APIs.
  • Optimización de rendimiento en aplicaciones web dinámicas.

Nivel (1-100): 80

Categoría: Junior

Descripción: Aprende a construir aplicaciones completas y persistentes que interactúan con APIs y almacenan datos. Este módulo es el puente hacia aplicaciones reales.

Módulo 7: Proyecto Final

Contenido:

  • Desarrollo de una aplicación completa desde cero.
  • Integración de todas las técnicas aprendidas: manipulación del DOM, APIs, y almacenamiento.
  • Pruebas y depuración del proyecto.
  • Preparación del proyecto para presentación o despliegue.

Nivel (1-100): 100

Categoría: Junior-Mid

Descripción: Integra todos los conocimientos adquiridos en un proyecto real y funcional que demuestre tus habilidades en JavaScript.
